# Projects

## PyWhisperer — AI-powered learning platform
**URL:** pywhisperer.dev
**Status:** Live, actively building under Lemitek

PyWhisperer is an AI-powered Django learning tool that generates personalised lessons tailored to the user's interests using LLMs. The idea came from a frustration with generic programming tutorials that feel disconnected from everyday life. PyWhisperer makes technical content feel personal — if you love football, your coding examples involve football data; if you love cooking, the examples involve recipes. This approach makes learning stickier and more engaging.

Toluwalemi built it using Python, Django, LangChain, React, and the OpenAI API. It is one of Lemitek's flagship projects. The platform is also expanding to teach AI engineering concepts, not just Django.

## Porosdjango — Django project bootstrapper (open-source)
**GitHub:** github.com/Toluwalemi/porosdjango
**Status:** Published on PyPI, 8 GitHub stars

Porosdjango is an open-source CLI tool that bootstraps Django projects with opinionated defaults. Setting up a new Django project from scratch involves a tedious amount of configuration — custom user models, REST framework setup, folder structure, environment handling, and more. Porosdjango automates all of that so a developer can go from zero to a properly configured project in seconds.

It was built using Python, Click (for the CLI interface), and Jinja2 for templating. It reflects Toluwalemi's preference for tools that remove friction from developer workflows.
